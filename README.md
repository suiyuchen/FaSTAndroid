FaSTAndroid
===========

FaSTAndroid是一个Android快速开发框架，目的是帮助大家快速高效开发Android应用。
类似的框架有很多，为什么要再多一个？这个框架的存在价值和意义是帮助你做出自己的框架，
了解框架设计者背后的思想和技术。

FaST框架会动态调整，大部分模块使用已经存在的开源项目。
框架本身保持灵活性，当有更好的模块出现，会被集成到此框架中。
